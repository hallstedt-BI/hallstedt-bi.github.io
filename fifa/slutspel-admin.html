Firebase Setup

  1. Gå till Firebase Console
  2. Klicka "Add project" och skapa ett nytt projekt
  3. Gå till Build → Realtime Database → Create Database
  4. Välj region (europe-west1) och "Start in test mode"
  5. Gå till Project Settings (kugghjulet) → General → Your apps → Web app (</>)
  6. Registrera en app och kopiera firebaseConfig objektet
  7. Klistra in värdena i firebase-config.js
